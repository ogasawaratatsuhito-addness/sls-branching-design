全体を貫く設計判断（v9準拠）

  1. 学習の中身（G2/G3/G4）→ 講師 ／ 手続き・トラブル（S1〜S4/X）→ サポート
  2. 人へ渡す時は必ず翻訳テンプレ4項目に整形（受講生に同じ説明を二度させない）
  3. 人へ繋ぐ瞬間は必ず「24時間以内に返事くるよ」
  4. 解決したら必ず「次のアクションに戻ろう」と学習へ送り出す
  5. トンマナ＝スキルプラスくん準拠。S3支払い・解約・不信だけ少し丁寧め
  6. LLMなしで将来AI化する枠は「回答スロット」（G2/G4の定型回答・S4の即答）として差し替え可能に実装
