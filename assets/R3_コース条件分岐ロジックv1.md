- 作成: 2026-05-30
- 対象: ゴール設定モード（質問項目-清書v2）の回答 → 次に歩む1コースを決める
- 出典: 2026-05-30 かいてぃーさん打ち合わせ＋たつ確定
- 前版: かいてぃーさん共有-コース条件分岐-v0.1 / コース分岐フロー-v0.6

## 大前提

- 提示するのは「次に歩む1コース」1つだけ。
- 「ビジネス入門コース」はクライアントワーク向けの土台コース。
- 自社（経営者・個人事業主）はビジネス入門に回さず、分野の自社向けコースへ直行する。

## 分岐に使う質問

- 問2 職業（属性）… 自社 / クライアントワーク の判定
- 問5 分野（9択）… どの系統のコースか
- 問6 ビジネス経験 … CWで入門を挟むかの判定（自社では見ない）
- 問7 生成AI経験 … 生成AI分野の入門/本コース出し分け
- 問8 PC使用状況 … PC前提ゲート
- 問9 学習時間 … コース選択には使わない（学習レポート用）

## 属性の定義

- 自社: 経営者・役員 / 個人事業主
- クライアントワーク（CW）: 会社員 / 公務員 / 学生 / パート・バイト / 主婦・主夫 / 休職中 / 求職中 / 退職・無職

## 分岐フロー（上から順に評価）

### ゲート1: PC前提（最優先）

- PCを持っていない、または 持っているが使い方がよく分かっていない
  → **ビジネス入門コース**
- それ以外 → ゲート2へ

### ゲート2: 属性

- 自社（経営者・個人事業主）→ **ゲート3A（経験年数は見ない）**
- クライアントワーク → **ゲート3B**

### ゲート3A: 自社 → 分野の自社向けコースへ直行

| 分野 | 案内コース |
|---|---|
| 生成AI | AI経験で分岐（※生成AIルール） |
| SNSマーケ | SNSマーケターコース |
| 広告マーケ | 広告マーケターコース |
| 導線構築 | 導線構築コース |
| デザイン | デザイナーコース |
| 動画編集 | 動画編集入門コース |
| ライティング | SEO入門コース |
| 営業 | 営業代行コース |
| マインドセット | マインドセットコース |

※自社はビジネス経験年数を見ない。3年未満でも分野コースへ直行（例: 自社×3年未満×SNS → SNSマーケター、自社×3年未満×動画 → 動画編集入門、自社×3年未満×生成AI → 生成AI入門直行）。

### ゲート3B: クライアントワーク → ビジネス経験で分岐

- **ビジネス経験 3年未満**
  → **ビジネス入門コース**（CW向けの土台）
  → 終了後に、選んだ分野のコースへ進む（生成AI分野の場合は「生成AI入門」から）
  → **例外: 動画編集を選んだ場合はビジネス入門を経由せず、動画編集コースへ直行**
- **ビジネス経験 3年以上**
  → 分野のCW向けコースへ直行（下表）

| 分野 | CW向けコース |
|---|---|
| 生成AI | AI経験で分岐（※生成AIルール） |
| SNSマーケ | Instagram運用代行コース |
| 広告マーケ | 広告運用代行コース |
| 導線構築 | 導線構築コース |
| デザイン | デザイナーコース |
| 動画編集 | 動画編集コース |
| ライティング | SEO入門コース |
| 営業 | 営業代行コース |
| マインドセット | マインドセットコース |

### 特例ルール: 生成AI分野は AI経験で入門/本コースを分ける

- AI経験なし → 生成AI入門コース
- AI経験あり（少しでも使ったことがある〜日常活用）→ 生成AIコース
- CW×3年未満で生成AIの場合は、ビジネス入門 → そのあと生成AI入門から。
